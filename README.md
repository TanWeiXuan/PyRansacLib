# PyRansacLib

Unofficial Python port of RansacLib

## Attribution

This project is a Unofficial Python only port of [RansacLib](https://github.com/tsattler/RansacLib), originally written in C++ by Torsten Sattler.

This port follows the structure, algorithms, and public API of the original project closely, but is implemented in Python.
